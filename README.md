# 🟦 CORE_STACK // Sistema de Apilamiento Cuántico

**CORE_STACK** es una reinvención moderna y minimalista de los juegos de lógica de bloques, envuelta en una atmósfera ciberpunk de alto contraste. Desarrollado con tecnologías web puras para un rendimiento óptimo.

## 🚀 Características Principales

* **Motor de Rotación Preciso:** Sistema de *Wall Kick* avanzado que evita errores de colisión en los bordes.
* **Dificultad Progresiva:** 20 niveles de velocidad incremental que pondrán a prueba tus reflejos.
* **Interfaz Neón:** Estética retro-futurista optimizada para pantallas OLED y modo oscuro.
* **Diseño Responsivo:** Jugable tanto en computadoras de escritorio (teclado) como en dispositivos móviles (panel táctil).
* **Gestión de Datos:** Visualización de "Próxima Pieza" y contador de puntuación en tiempo real.

## 🎮 Controles de Mando

| Acción | Teclado | Panel Táctil |
| :--- | :--- | :--- |
| **Mover Izquierda/Derecha** | Flechas `←` `→` | Botones Púrpura |
| **Rotación Horaria** | Flecha `↑` | Botón Cyan (Derecho) |
| **Rotación Anti-horaria** | Tecla `Z` | Botón Cyan (Izquierdo) |
| **Caída Suave** | Flecha `↓` | Botón Púrpura Abajo |
| **Caída Rápida (Drop)** | Espacio `Space` | Botón Rosa (Drop) |

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura y animaciones de neón con variables CSS.
* **JavaScript (Vanilla):** Lógica del motor de juego y sistema de colisiones.
* **Bootstrap 5:** Estructura de rejilla (grid) responsiva.
* **Google Fonts:** Tipografía monoespaciada para estilo técnico.
